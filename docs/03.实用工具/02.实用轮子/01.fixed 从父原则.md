## 什么是从父原则

- 子元素在与父辈元素比较的时候，用父辈的 z-index 去比较
- 在与同级元素比较的时候，才有自己的 z-index 去比较

## 什么情况下出现从父原则

- 父元素通过 fixed absolute relative 定位的元素, 子元素也是 fixed absolute relative 定位的元素。

- 在父元素上设置了 z-index

- 跟父元素同级的元素也是通过 fixed absolute relative 定位的元素，并且设定了 z-index
