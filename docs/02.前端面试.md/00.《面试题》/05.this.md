## 面试题

```js
var number = 5;
var obj = {
    number: 3,
    fn1: (function () {
        var number;
        this.number *= 2;
        number = number * 2;
        number = 3;
        return function () {
            var num = this.number;
            this.number *= 2;
            console.log(num);
            number *= 3;
            console.log(number);
        }
    })()
}
var fn1 = obj.fn1;
fn1.call(null);
obj.fn1();
console.log(window.number);

```
## this是什么？

this 就是一个指针，指向调用函数的对象

**this的绑定规则**

- 默认绑定
- 隐式绑定
- 硬绑定
- new绑定
